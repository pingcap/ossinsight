## 竞品分析

当前开源分析工具的商业化模式：

| 竞品 | 免费层 | 付费功能 | 定价 |
|------|-------|---------|------|
| Libraries.io | 基础搜索 | API 限额提升 | $29-299/月 |
| Sourcegraph | 开源版 | 企业代码搜索 | $8/用户/月 |
| Snyk | 基础安全扫描 | 高级安全功能 | $25-99/月 |
| **OSSInsight** | 全功能 | ❌ | ❌ |

## 现状问题

OSSInsight 目前完全免费，依赖 PingCAP 赞助。长期可持续性存疑。

## 建议功能

### 1. 企业版功能 (Enterprise Tier)

#### 私有 Repo 分析
- 连接企业 GitHub Organization
- 私有 repo 健康度监控
- 团队贡献分析
- 内部 benchmark 对比

#### 团队 Dashboard
- 多项目聚合视图
- 团队生产力指标
- 跨 repo 趋势分析
- 自定义报告导出

#### SLA 监控
- Issue/PR 响应时间追踪
- 自定义 SLA 目标
- 违约预警通知
- 月度报告

### 2. API 付费层

当前 API 完全免费，建议分级：

| 层级 | 请求限额 | 价格 |
|------|---------|------|
| Free | 100/天 | $0 |
| Pro | 10,000/天 | $49/月 |
| Enterprise | 无限制 | 定制 |

### 3. 数据导出服务

- 原始数据 CSV/JSON 导出
- 自定义时间范围
- 批量导出 API
- 定期自动导出

### 4. 白标报告

- 去除 OSSInsight branding
- 自定义品牌报告
- 嵌入企业内部系统
- API 集成

## 定价建议

参考竞品和 TiDB Cloud 定价策略：

- **Pro**: $49/月 - 适合个人开发者/小团队
- **Team**: $199/月 - 适合中型团队（5-10 人）
- **Enterprise**: 定制 - 大型企业

## 风险提示

⚠️ **社区反弹**: 开源社区可能对商业化敏感
⚠️ **竞品免费**: GitHub 原生功能也在增强

## 缓解策略

1. **保持核心功能免费**: Repo/Developer 分析继续免费
2. **透明沟通**: 明确说明成本结构
3. **开源贡献者优惠**: 活跃维护者折扣
4. **教育优惠**: 学生/研究者免费

## 预期收入

保守估计（上线 12 个月后）：
- Pro: 100 用户 × $49 = $4,900/月
- Team: 20 团队 × $199 = $3,980/月
- Enterprise: 5 客户 × $2000 = $10,000/月
- **总计**: ~$19,000/月
